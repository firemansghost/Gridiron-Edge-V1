# Gridiron Edge Web App

Next.js web application for Gridiron Edge college football analytics platform.

## Tech Stack
- Next.js 14 (App Router)
- Tailwind CSS
- shadcn/ui components
- Recharts for data visualization
- TypeScript

## Development
```bash
npm install
npm run dev
```

## Deployment
Deployed on Vercel with environment variables configured in Vercel dashboard.
